How to Use ?

Home Page

  • Home page shows the bookmarked faviourite locations
  • MapIcon for choose the faviourite location and add to bookmarks.
  • Help for instructions to use application
  • Settings for change the Units, Date format and reset bookmark locations.
  • When click on location, It moves to Weather details page.

Manage Bookmark Locations

  • To add location, Long press 2sec on mapview at required place and confirm it.
  • By Swipe left, you can clear the particular location
  • We can reset the all bookmark locations from settings -> Reset bookmark locations option.
  • Bookmark locations will be stored for next time usage of application. It keeps the data until the app is uninstalled.

Settings

  • Units -> It allows to choose Metric/ Imperial units. These settings will be saved for next time use of application.
  • Selected Units will be reflect on weather information page.
  • Date Formats -> To select the required date format and it will be save in storage.
  • Selected date format will be reflect on weather information page.
  • Reset Bookmark locations to clear the previous saved locations

Weather Information

  • It shows the today temparature, Humidity and wind speed.
  • It shows the 5 days forecast temparature, Humidity, Wind speed & Rain Chances
